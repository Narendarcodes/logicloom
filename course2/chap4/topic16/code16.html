String-Handling Functions

  • In C, string handling is commonly done using functions from the library.
  • Some essential functions include:
  • strlen(str):
    Returns the length of the string str.
  • strcpy(dest, src):
    Copies the string src to dest. strcat(dest, src): Concatenates the string src onto the end of dest.
  • strcmp(str1, str2):
    Compares two strings str1 and str2.
  • strchr(str, ch):
    Returns a pointer to the first occurrence of character ch in string str.
  • strstr(str1, str2):
    Returns a pointer to the first occurrence of string str2 in string str.
