Introduction

  • In C, an array is a collection of elements of the same data type stored in contiguous memory locations.
  • It provides a way to store and manipulate a fixed-size sequence of elements.
  • The declaration syntax is:
    datatype arrayName[size];
  • For example, to declare an integer array with 5 elements:
    int myArray[5];
  • Arrays are zero-indexed, meaning the first element is accessed with index 0, the second with index 1, and so on.
  • You can initialize an array during declaration:
    int myArray[5] = {1, 2, 3, 4, 5};
  • Or assign values later:
    myArray[0] = 10;
    myArray[1] = 20;
