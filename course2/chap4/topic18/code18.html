Other Features of Strings

  • In C, strings are represented as arrays of characters, and there are several features and functions associated with working with strings.
  • Here are some key aspects:
  • Null-terminated:
    C strings are null-terminated, meaning they end with a null character ('\0'). This character is used to indicate the end of the string.
  • String functions:
    C provides standard library functions for string manipulation, such as strlen (to find the length), strcpy (to copy strings), strcat (to concatenate strings), and many more.
  • Array of characters:
    Strings are essentially arrays of characters, and individual characters in a string can be accessed using array notation.
  • String input/output:
    You can use printf and scanf functions to input and output strings. The %s format specifier is commonly used for strings.
  • String literals:
    Strings can be represented as string literals, enclosed in double quotes. For example: "Hello, World!".
