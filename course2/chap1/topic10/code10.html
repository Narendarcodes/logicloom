Pseudocode

  • Pseudocode is a high-level description of an algorithm that uses a mixture of natural language and some programming language-like constructs.
  • It helps in planning and understanding the logic of an algorithm before it is implemented in a specific programming language.
  • Pseudocode is not tied to the syntax of any particular programming language, making it a versatile tool for algorithm design.
    • Example :

    Procedure Add_Two_Numbers(A, B)
    Sum = A + B
    Return Sum
    End Procedure

  • Add Two Numbers is procedure(function) name.
  • A and B are the inputs representing the two numbers to added.
  • Sum is variable that store the result of adding of two numbers and then return the sum.
